Events | Dev Initiative

Dev Initiative is a new program being implemented this semester. The goal is to provide more structure and mentorship to semester-long projects, for those who want it.

Below is a list of current project ideas with volunteered project leaders. If you have your own project idea, or have not voted, please do so on our Facebook group.

  • Web Dev Crash Course
    • Overview for basic HTML/CSS, eventually javascript libraries and backends
    • How to obtain and maintain domains
  • Python Web Scraper
    • How to locally save and use data from websites with Python
    • Use of Python libraries (Beautiful Soup, Selenium, etc.)
  • DIY SmartMirror / Motion Tracking Nerf Gun
    • Modding hardware to communicate with microcomputers (Raspberry Pi / Arduino)
    • Use of handtools
  • Facebook Messenger Bot
    • Building FAQ bot on Facebook Messenger app to help with our Spring Hackathon
    • Use of Python, hosted on Heroku
  • Oculus Rift VR
    • Game/app development for Oculus
    • Use of Unity and Blender, but open to suggestions
  • The Infirmary
    • Diagnosing/repairing club-owned hardware
    • 3D printer, drone, LED board
